about=VertexWarp provides an interactive workflow for refining georeferenced imagery:
    - Side-by-side synchronized views (reference basemap vs target image)
    - Manual vertex adjustment with topology-preserving drag
    - Iterative refinement using Polynomial or TPS warping
    - Target accuracy: sub-meter (0.5m or better)

    Workflow:
    1. Load target image and known-good field boundary
    2. Drag boundary vertices to match actual positions on target
    3. Click Refine to warp target back to original boundary
    4. Repeat until satisfied, then export
